Question-1: What are the different ways to select an element in the DOM?

DOM থেকে কোনো এলিমেন্ট সিলেক্ট করতে চাইলে "getElementById()" দিয়ে ID অনুযায়ী "getElementsByClassName()" দিয়ে class name অনুযায়ী, আর "getElementsByTagName()" দিয়ে tag name অনুযায়ী সিলেক্ট করা যায়। "querySelector()" দিয়ে প্রথম মিল থাকা এলিমেন্টটা নিয়ে আসা যায়, আর "querySelectorAll()" দিয়ে সব মিল থাকা এলিমেন্ট নিয়ে আসা যায়। কোনো এলিমেন্টের প্যারেন্ট পেতে "parentNode" আর তার চাইল্ড এলিমেন্ট পেতে "children" ইউজ করা হয়। সবশেষে, "closest()" ইউজ করে নির্দিষ্ট সিলেক্টরের সাথে মিল থাকা সবচেয়ে কাছের প্যারেন্ট খুঁজে বের করা যায়।

Question-2: What is the difference between innerHTML, innerText, and textContent?

"innerHTML" দিয়ে কোনো এলিমেন্টের HTML কন্টেন্ট ট্যাগ সহ পাওয়া বা সেট করা যায়। আর "innerText" দিয়ে কোনো ট্যাগের ভিতরের টেক্সট পায় বা সেট করা যায়, আর "textContent" সম্পর্কে তেমন কিছু জানি না

Question-3: What is event delegation in the DOM?

DOM এ Event Delegation মানে হলো প্রতিটি চাইল্ড এলিমেন্টের জন্য আলাদা ইভেন্ট লিসেনার না দিয়ে একটা parent এলিমেন্টে ইভেন্ট লিসেনার যোগ করা। যখন কোনো চাইল্ড এলিমেন্টে ইভেন্ট ট্রিগার হয় তখন সেটা parent পর্যন্ত পৌঁছে যায় event bubbling এর কারনে।

Question-4: What is event bubbling in the DOM?

Event Bubbling মানে হলো কোনো ইভেন্ট যখন একটা এলিমেন্টে ঘটে তখন সেটা তার প্যারেন্টদের কাছে পৌঁছে যায়। যেমন যদি একটা button এর ভেতর ক্লিক করি ইভেন্টটা প্রথমে button এ হবে তারপর div এ তারপর body তে শেষে document পর্যন্ত পৌঁছাবে।

Question-5: How do you create, add, and remove elements using JavaScript?

JavaScript এ নতুন এলিমেন্ট বানাতে "document.createElement('tagName')" ব্যবহার করা হয় "tagName" যেকোনো HTML এলিমেন্ট হতে পারে যেমন div, p, span ইত্যাদি। নতুন এলিমেন্ট যোগ করতে "parentElement.appendChild(newElement)" ব্যবহার করা হয়। কোনো এলিমেন্ট রিমুভ করতে "parentElement.removeChild(elementToRemove)" ব্যবহার করা হয় অথবা সরাসরি "element.remove()" দিলেও কাজ হবে।
