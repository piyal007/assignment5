Question-1: What are the different ways to select an element in the DOM?
To select an element in the DOM, we can use `getElementById()` to select by ID, `getElementsByClassName()` to select by class name, and `getElementsByTagName()` to select by tag name. `querySelector()` selects the first matching element, and `querySelectorAll()` selects all matching elements. We can also use `parentNode` to get the parent element or `children` to get child elements. Lastly, `closest()` finds the nearest ancestor that matches a selector.
Question-2: What is the difference between innerHTML, innerText, and textContent?
`innerHTML` gets or sets the HTML content of an element, including tags. `innerText` gets or sets the visible text content, respecting styles like hidden elements. `textContent` gets or sets all the text inside an element, without considering styles or hidden content, and doesn't include HTML tags.
Question-3: What is event delegation in the DOM?
Event delegation in the DOM is a technique where we attach a single event listener to a parent element instead of attaching one to each child element. The parent listens for events on its child elements and handles them when they occur. This works because of event bubbling where the event starts at the target element and bubbles up to its parent. This method makes it easier to manage events especially when elements are added or removed dynamically.
Question-4: What is event bubbling in the DOM?
Event bubbling in the DOM happens when an event starts at the element where it occurred and then moves up to its parent elements. For example if we click on a button inside a div the event will first happen on the button then move up to the div then to the body and finally to the document. This allows parent elements to catch events from their child elements making it easier for us to manage events.
Question-5: How do you create, add, and remove elements using JavaScript?
To create an element in JavaScript we use document.createElement('tagName') where 'tagName' can be any HTML element like div, p, or span. Once we have the new element we can add it to the page by using parentElement.appendChild(newElement). To remove an element we use parentElement.removeChild(elementToRemove) to remove it from its parent. Another way is by calling element.remove() directly on the element to remove it.
